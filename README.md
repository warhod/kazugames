# Kazu Games

**Kazu Games** is a personalized game tracker and collection manager for Nintendo Switch: build your library, organize friend groups, and coordinate lending — without positioning the product as a “deals” site.

## Key Features

- **Collection Tracking**: Keep a curated list of all your games
- **Catalog & prices (via Deku)**: See list prices and history from the DekuDeals catalog
- **Community Sharing**: Organize and share your collection with friends or groups.
- **Premium Design**: Built with a modern, immersive aesthetic to provide a high-quality user experience.
